# CSCN71030-Sec2--Group4 Project

## Overview
This project is a simple budget-based search and recommendation system.  
The user enters a category and a budget, and the system returns matching facilities and suggests the best options.

---

## Features
- Search by category  
- Filter results based on budget  
- Generate recommendations  
- Sort results by price  
- Rank results by rating  
- Display final results  

---

## Modules
The system is divided into main parts such as:
- Main Control Module (controls the program flow)
- Search and Filtering Module (filters results based on input)
- Recommendation Module (generates final results)

There are also other supporting modules for things like sorting, storing data, and displaying results.

---

## Testing
- Unit testing was done using Visual Studio  
- Integration testing was done across modules  
- All test cases passed  

---

## How to Run
1. Open the project in Visual Studio  
2. Build the solution  
3. Run the program  
4. Enter category and budget when asked
